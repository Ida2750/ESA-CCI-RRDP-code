# Plotting Scripts Overview

This repository contains plotting scripts and input data used to generate the figures in the associated paper (A first approach towards dual-hemisphere sea ice reference measurements from multiple data sources repurposed for evaluation and product intercomparison of satellite altimetry): #INSERT DOI WHEN AVAILABLE

## Data Files

- **IMB_info_table_Henriette.dat**
- **SIMBA_initial_thickness.dat**

These files are used as input data to create **Figure 3**, which is generated using the script `plot_IMBs_oneColor.bat`.
